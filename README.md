## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/sevanial/sevanial.net.git
cd sevanial.net

# Serve with any static server
python -m http.server 8000
# or
php -S localhost:8000

# Visit in browser
open http://localhost:8000
```

**No build process required** - Just clone and serve!

## ⚙️ Configuration

### Content Management
Edit the configuration variables in `index.php`:

```php
// Status updates
$now_status = "Your current activity";
$last_updated = "Current date";
$next_stream = "Your schedule";

// Minecraft server
$mc = [
    'ip' => 'your-server.com',
    'version' => '1.21.7',
    'type' => 'Server type',
    'status' => 'Online/Offline'
];

// Social links
$social = [
    'youtube' => 'https://youtube.com/@yourhandle',
    'twitch' => 'https://twitch.tv/yourhandle',
    // ... more platforms
];
```

### Theme Customization
Modify CSS variables in `styles.css`:

```css
:root {
  --bg: #0a0a0a;           /* Background color */
  --accent: #6366f1;       /* Primary accent */
  --accent2: #8b5cf6;      /* Secondary accent */
  /* ... more variables */
}

[data-theme="light"] {
  --bg: #fff;              /* Light theme overrides */
  /* ... light theme colors */
}
```

## 🎯 Required HTML Structure

For JavaScript functionality to work properly, ensure these elements exist:

| Component | Required ID/Class | Purpose |
|-----------|------------------|---------|
| Theme toggle | `#theme-btn` | Dark/light mode switching |
| Mobile menu | `#menu-btn`, `#nav` | Responsive navigation |
| Header | `#header` | Scroll-aware header behavior |
| Back to top | `#back-to-top` | Smooth scroll to top |
| Server IP | `#server-ip` | Minecraft server display |
| Copy button | `.copy-btn` | Clipboard functionality |
